너 rnd 함수가 지금 틀린 이유(핵심만)

1) 너 rnd는 “정수”를 반환하지 않음
- x가 2.2면 그대로 2.2를 리턴함
- x가 2.7이면 3.7을 리턴함 (x+1이라서 2.7 -> 3.7)
- 즉, 반올림이 아니라 “실수에 1을 더한 실수”가 되어버림

2) 0.5 처리도 요구사항과 다름
- 보통 이 문제는 0.5면 올림인데,
- 너는 x%1 > 0.5만 올림이라서
  x%1 == 0.5일 때는 올림이 안 됨

3) cut은 인덱스로 써야 해서 반드시 int여야 함
- cut이 실수면 슬라이싱에서 에러/오동작이 남

---

정확한 반올림(0.5는 올림) 함수

```python
def rnd(x):
    return int(x + 0.5)
